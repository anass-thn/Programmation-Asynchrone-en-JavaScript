# TP - Programmation Asynchrone en JavaScript

Ce dépôt contient une série d'exercices pratiques pour maîtriser la programmation asynchrone en JavaScript.

## Contenu

Le fichier `exercices-js-async.html` regroupe plusieurs concepts clés :

1.  **Callbacks & setTimeout** : Comprendre l'asynchronisme de base.
2.  **Gestion d'erreurs** : Le pattern "Error-first callback".
3.  **AJAX (XMLHttpRequest)** : Faire des requêtes HTTP à l'ancienne.
4.  **Callback Hell** : Visualiser le problème des callbacks imbriqués.
5.  **Promesses (Promises)** : Chaînage et gestion plus propre.
6.  **Fetch API** : La méthode moderne pour les requêtes réseau.
7.  **Promise.all** : Exécuter des tâches en parallèle.
8.  **Async / Await** : La syntaxe moderne pour l'asynchrone.
9.  **Dashboard Temps Réel** : Application finale combinant tous les concepts.

## Comment utiliser

1.  Téléchargez ou clonez ce dépôt.
2.  Ouvrez le fichier `exercices-js-async.html` dans votre navigateur.
3.  Ouvrez le fichier dans un éditeur de code (VS Code) pour voir et modifier les solutions.
4.  Utilisez les boutons sur la page pour tester chaque exercice.
